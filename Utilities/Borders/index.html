Borders

Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

Basic example

Use border utilities to add or remove an elements borders.

Additive

Additive classes are used to add borders.

      <span class="Border"></span>
      <span class="Border-top"></span>
      <span class="Border-end"></span>
      <span class="Border-bottom"></span>
      <span class="Border-start"></span>
Subtractive

Subtractive classes are used to remove borders.

    <span class="Border Border-0"></span>
    <span class="Border Border-top-0"></span>
    <span class="Border Border-end-0"></span>
    <span class="Border Border-bottom-0"></span>
    <span class="Border Border-start-0"></span>
Border-width
    <span class="Border Border-1"></span>
    <span class="Border Border-2"></span>
    <span class="Border Border-3"></span>
    <span class="Border Border-4"></span>
    <span class="Border Border-5"></span>

Change the border color using utilities built on our theme colors.

Sizes

Use the scaling classes for larger or smaller rounded corners. Sizes range from 0 to 5, and can be configured by modifying the utilities API. 

    <span class="Bg-primary Rounded Rounded-1"></span>
    <span class="Bg-primary Rounded Rounded-2"></span>
    <span class="Bg-primary Rounded Rounded-3"></span>
    <span class="Bg-primary Rounded Rounded-4"></span>
    <span class="Bg-primary Rounded Rounded-5"></span>
Sass - Variables
  $Border-width   : 1px !default;
  $Border-widths  : (0: 0, 1: 1px, 2: 2px, 3: 3px, 4: 4px, 5: 5px) !default;

  $Rounded        : .25rem !default;

  $Rounded-sm     : .20rem !default;
  $Rounded-md     : .375rem !default;
  $Rounded-lg     : .75rem !default;
  $Rounded-xl     : 1.0rem !default;
  $Rounded-xxl    : 1.5rem !default;

  $Rounded-pill   : 50rem !default;
  $Rounded-circle : 50% !default;
Sass - Utilities API

Border utilities are declared in our utilities API in Utils/_border.scss.


$utilities-Border: (
  // Border
        "Border": (
                Class: Border, Properties: border, Important: true,
                Values: (null: $Border-width solid $Border-color, 0: 0),
        ),
  // Border Directions
        "Border-top": (
                Class: Border-top, Properties: border-top, Important: true,
                Values: (null: $Border-width solid $Border-color, 0: 0),
        ),
        "Border-right": (
                Class: Border-end, Properties: border-right, Important: true,
                Values: (null: $Border-width solid $Border-color, 0: 0),
        ),
        "Border-bottom": (
                Class: Border-bottom, Properties: border-bottom, Important: true,
                Values: (null: $Border-width solid $Border-color, 0: 0),
        ),
        "Border-left": (
                Class: Border-start, Properties: border-left, Important: true,
                Values: (null: $Border-width solid $Border-color, 0: 0),
        ),
  // Border-width
        "Border-width": (
                Class: Border, Properties: border-width, Important: true,
                Values: $Border-widths,
        ),
  // Rounded
        "Rounded": (
                Class: Rounded, Properties: border-radius, Important: true,
                Values: (
                        null: #{var(--#{$var-Prefix}border-radius, $Rounded)},
                        0: 0, circle: $Rounded-circle, pill: $Rounded-pill
                ),
        ),
  // Scale
        "Rounded-scale": (
                Class: Rounded, Properties: --#{$var-Prefix}border-radius, Important: false,
                Values: (1: $Rounded-sm, 2: $Rounded-md, 3: $Rounded-lg, 4: $Rounded-xl, 5: $Rounded-xxl),
        ),
  // Rounded Directions
        "Rounded-top": (
                Class: Rounded-top, Properties: border-top-left-radius border-top-right-radius, Important: false,
                Values: (null: #{var(--#{$var-Prefix}border-radius, $Rounded)}),
        ),
        "Rounded-top-0": (
                Class: Rounded-top, Properties: border-top-left-radius border-top-right-radius, Important: true,
                Values: (0: 0),
        ),
        "Rounded-right": (
                Class: Rounded-end, Properties: border-top-right-radius border-bottom-right-radius, Important: false,
                Values: (null: #{var(--#{$var-Prefix}border-radius, $Rounded)}),
        ),
        "Rounded-right-0": (
                Class: Rounded-end, Properties: border-top-right-radius border-bottom-right-radius, Important: true,
                Values: (0: 0),
        ),
        "Rounded-bottom": (
                Class: Rounded-bottom, Properties: border-bottom-right-radius border-bottom-left-radius, Important: false,
                Values: (null: #{var(--#{$var-Prefix}border-radius, $Rounded)}),
        ),
        "Rounded-bottom-0": (
                Class: Rounded-bottom, Properties: border-bottom-right-radius border-bottom-left-radius, Important: true,
                Values: (0: 0),
        ),
        "Rounded-left": (
                Class: Rounded-start, Properties: border-bottom-left-radius border-top-left-radius, Important: false,
                Values: (null: #{var(--#{$var-Prefix}border-radius, $Rounded)}),
        ),
        "Rounded-left-0": (
                Class: Rounded-start, Properties: border-bottom-left-radius border-top-left-radius, Important: true,
                Values: (0: 0),
        ),
  // Subtractive 1 Side
        "Rounded-top-left-0": (
                Class: Rounded-tl, Properties: border-top-left-radius, Important: true, Values: (0: 0)
        ),
        "Rounded-top-right-0": (
                Class: Rounded-tr, Properties: border-top-right-radius, Important: true, Values: (0: 0)
        ),
        "Rounded-bottom-right-0": (
                Class: Rounded-br, Properties: border-bottom-right-radius, Important: true, Values: (0: 0)
        ),
        "Rounded-bottom-left-0": (
                Class: Rounded-bl, Properties: border-bottom-left-radius, Important: true, Values: (0: 0)
        ),
);

@include Utilities($utilities-Border);
