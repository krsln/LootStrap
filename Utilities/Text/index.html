Word break

Prevent long strings of text from breaking your components layout by using .Text-break to set word-wrap: break-word and word-break: break-word. We use word-wrap instead of the more common overflow-wrap for wider browser support, and add the deprecated word-break: break-word to avoid issues with flex containers.

mmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmm

    <p class="Text-break">mmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmm</p>

Font size

Quickly change the font-size of text. While our heading classes (e.g., .H1.H6) apply font-size, font-weight, and line-height, these utilities only apply font-size. Sizing for these utilities matches HTMLs heading elements, so as the number increases, their size decreases.

.FS-1 text

.FS-2 text

.FS-3 text

.FS-4 text

.FS-5 text

.FS-6 text

    <p class="FS-2">.FS-2 text</p>

    <p class="FS-4">.FS-4 text</p>
    <p class="FS-5">.FS-5 text</p>
    <p class="FS-6">.FS-6 text</p>

Maps

Font-size utilities are generated from this map, in combination with our utilities API.

$font-size-base: 1rem !default; // Assumes the browser default, typically `16px`

$h1-font-size: $font-size-base * 2.5 !default;
$h2-font-size: $font-size-base * 2 !default;
$h3-font-size: $font-size-base * 1.75 !default;
$h4-font-size: $font-size-base * 1.5 !default;
$h5-font-size: $font-size-base * 1.25 !default;
$h6-font-size: $font-size-base !default;

$font-sizes: (
        1: $h1-font-size,
        2: $h2-font-size,
        3: $h3-font-size,
        4: $h4-font-size,
        5: $h5-font-size,
        6: $h6-font-size
) !default;
Sass - Utilities API

Font and text utilities are declared in our utilities API in Utils/_text.scss.

$utilities-Text: (
  // Font
        "Font-family": (
                Class: Font, Properties: font-family, Important: true,
                Values: (monospace: var(--#{$var-Prefix}font-monospace))
        ),
        "Font-size": (
                Class: FS, Properties: font-size, Important: true,
                Values: $font-sizes
        ),
        "Font-style": (
                Class: FST, Properties: font-style, Important: true, Values: italic normal
        ),
        "Font-weight": (
                Class: FW, Properties: font-weight, Important: true,
                Values: (
                        light: $font-weight-light,
                        lighter: $font-weight-lighter,
                        normal: $font-weight-normal,
                        bold: $font-weight-bold,
                        bolder: $font-weight-bolder
                )
        ),
  // Line
        "Line-height": (
                Class: LH, Properties: line-height, Important: true,
                Values: (1: 1, sm: $line-height-sm, base: $line-height-base, lg: $line-height-lg,)
        ),
  // Text
        "Text-align": (
                Class: Text, Properties: text-align, Important: true, Responsive: true,
                Values: (start: left, center: center, end: right)
        ),
        "Text-decoration": (
                Class: Text-decoration, Properties: text-decoration, Important: true,
                Values: (none, underline, line-through, overline,)
        ),
        "Text-transform": (
                Class: Text, Properties: text-transform, Important: true,
                Values: lowercase uppercase capitalize
        ),
        "White-space": (
                Class: Text, Properties: white-space, Important: true,
                Values: (wrap: normal, nowrap: nowrap,)
        ),
        "Word-wrap": (
                Class: Text, Properties: word-wrap word-break, Important: true, // rtl: false
                Values: (break: break-word)
        ),
  // Color => for text colors, muted and reset check _color.scss
);

@include Utilities($utilities-Text);
